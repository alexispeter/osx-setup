osx-setup
=========

Setup OS X

## OSX
1. Erase disk "Case-sensitive, Journaled" with "Disk Utility"
2. Install OSX
3. Encrypt disk (FileVault)

## Migration
- Copy your `~/.ssh/`?

## Repository
Clone this repository `https://github.com/cargomedia/osx-setup.git`.

## System
Run the install script and reboot.
```bash
./install.sh
sudo reboot
```
