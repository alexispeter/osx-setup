osx-setup
=========

Setup OS X

## OSX
1. Erase disk "Case-sensitive, Journaled" with "Disk Utility"
2. Install OSX
3. Encrypt disk (FileVault)

## Migration
- Copy your `~/.ssh/`?

## System
Run the installation script.
```bash
curl -Ls https://raw.github.com/cargomedia/osx-setup/master/osx-install.sh | bash
```
